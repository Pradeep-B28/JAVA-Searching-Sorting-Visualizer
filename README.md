## ✨ Features

- ⚡ **Step-by-Step Visualization** – Watch every swap, comparison, and search iteration in real-time.
- 🎛️ **Adjustable Speed** – Slow down to understand the logic or speed up for quick demos.
- 🧩 **Multiple Algorithms** – Covers all the essential sorting and searching techniques.
- 🎲 **Random Array Generation** – Generate new test data with a single click to test different cases.
- 📊 **Comparison Metrics** – Track time complexity and number of operations performed.
